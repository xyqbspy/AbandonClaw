## ADDED Requirements

### Requirement: 新增学习动作必须先按动作语义复用既有按钮层级
系统 MUST 在新增学习页面、功能入口或动作按钮时，先根据动作语义判断它属于主动作、次动作、危险动作还是图标动作，并复用既有按钮层级和组件封装，不得为单个页面临时创造新的主色、边框、底色或按钮语义。

#### Scenario: 维护者新增学习动作按钮
- **WHEN** 维护者在 `today`、`scenes`、`scene`、`lesson`、`chunks`、`review` 或其它学习链路页面中新增动作按钮
- **THEN** 该按钮必须先按动作语义映射到既有主按钮、次按钮、危险按钮或图标按钮层级
- **AND** 若动作语义与已有页面相同，视觉层级必须保持一致
- **AND** 若无法映射到既有层级，必须先记录新增层级的必要性和影响范围，不得直接在页面内手拼一次性样式
