## MODIFIED Requirements

### Requirement: Today 必须作为每日学习入口
系统 MUST 提供 `today` 入口，把继续学习、复习任务和学习概览聚合成当日可执行入口。

#### Scenario: 用户进入今日页
- **WHEN** 用户打开 `today`
- **THEN** 页面应展示继续学习入口、复习相关信息或学习概览中的至少一类核心信息
- **AND** 这些信息应来自学习链路的聚合结果，而不是孤立静态展示
- **AND** `today` 页面中继续学习卡片、今日任务和相关说明文案必须基于稳定的数据映射契约来解释用户当前学习步骤与完成状态
- **AND** 当服务端继续学习聚合结果缺失时，前端只能按约定使用本地 repeat 状态或场景列表作为兜底入口，而不得重新定义新的学习完成语义
