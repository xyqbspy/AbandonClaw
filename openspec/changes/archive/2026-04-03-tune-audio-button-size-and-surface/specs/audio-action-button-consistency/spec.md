# audio-action-button-consistency

## MODIFIED Requirements

### Requirement: 音频动作按钮必须默认以纯 icon 形式展示
系统 MUST 将前端音频动作按钮统一为纯 icon 视觉形态，避免在不同页面同时存在 icon-only 与 icon+文本两套并行样式。系统还 MUST 允许页面在不破坏统一组件的前提下，为纯 icon 按钮声明与当前界面一致的尺寸和表面色样式。

#### Scenario: 页面渲染朗读或循环播放入口
- **WHEN** `lesson`、`scene detail`、`chunks`、selection detail 等页面渲染音频动作按钮
- **THEN** 按钮正文不得再显示“朗读”“播放”“循环播放”等可见文本
- **AND** 按钮必须以统一 icon 样式呈现
- **AND** 页面如需适配当前视觉上下文，必须基于统一音频按钮组件调整尺寸或表面色，而不是重新实现独立按钮

### Requirement: 音频动作按钮状态变化不得依赖可见文本
系统 MUST 允许音频动作按钮在 active 或 loading 状态下继续表达状态变化，但这种表达不得依赖重新显示正文文字。系统还 MUST 让状态按钮在不同页面语义表面上保持协调，不得因为进入播放态或加载态而出现与所在动作区割裂的底色。

#### Scenario: 按钮处于播放中或加载中
- **WHEN** 某个音频动作按钮进入 active 或 loading 状态
- **THEN** 按钮必须仍保持纯 icon 展示
- **AND** 状态反馈必须通过 icon 样式、动画和可访问名称完成
- **AND** 按钮表面色必须继续与当前页面动作区保持协调

## ADDED Requirements

### Requirement: 对话气泡下方动作按钮必须保持同组视觉规格
系统 MUST 让对话气泡下方的翻译按钮和播放按钮表现为同一组辅助动作，避免出现一大一小或背景层级不一致的情况。

#### Scenario: 用户查看句子气泡下方动作区
- **WHEN** 页面在句子气泡下方同时展示翻译入口和播放入口
- **THEN** 两个按钮必须使用一致的点击区尺寸与圆角规格
- **AND** 播放按钮必须与翻译按钮保持统一的背景层级
- **AND** 默认态、播放中和加载中的按钮背景不得再使用与当前动作区不协调的白色表面

### Requirement: 详情区朗读按钮必须复用统一按钮表面规则
系统 MUST 让 selection detail 中的主句朗读按钮和相关短语朗读按钮复用统一音频按钮组件，避免详情区继续维护独立的图标按钮实现。

#### Scenario: 用户查看 selection detail 中的朗读入口
- **WHEN** 页面在 selection detail 面板或 sheet 中渲染主句朗读按钮、相关短语朗读按钮
- **THEN** 这些按钮必须基于统一音频按钮组件渲染
- **AND** 按钮必须与详情区当前表面层级保持协调
- **AND** 不得继续保留仅在详情区生效的独立音频按钮封装

### Requirement: 阅读正文中的纯 icon 朗读按钮必须保持统一尺寸层级
系统 MUST 让句子列表、段落阅读块等正文区域中的纯 icon 朗读按钮与已统一的详情区 / 气泡动作区保持一致的尺寸层级，避免同一页面里继续混用明显更小的一档按钮规格。

#### Scenario: 用户查看正文列表中的朗读入口
- **WHEN** 页面在 `sentence block`、非对话 `lesson reader` 等正文阅读区域渲染纯 icon 朗读按钮
- **THEN** 按钮必须使用统一音频按钮组件的标准 icon 尺寸层级
- **AND** 页面如为卡片或轻表面场景，按钮应继续与所在表面保持协调

### Requirement: 纯 icon 朗读按钮不得在部分页面继续保留独立凸起底板
系统 MUST 让 `chunks` 等页面中的纯 icon 朗读按钮与当前统一样式方向保持一致；当页面已经采用“按钮贴合背景层”的视觉规则时，不得在局部重新引入独立描边和凸起底板。

#### Scenario: 用户查看 chunks 详情或例句卡片里的朗读入口
- **WHEN** 页面在 `focus detail` 顶部、例句卡片等区域渲染纯 icon 朗读按钮
- **THEN** 按钮必须继续复用统一音频按钮组件
- **AND** 不得单独使用与所在表面割裂的描边或凸起底板

#### Scenario: 用户查看头部、工具栏或列表中的纯 icon 音频按钮
- **WHEN** 页面在 `lesson reader` 头部、`selection toolbar`、`chunks list` 等区域渲染纯 icon 的朗读或循环按钮
- **THEN** 这些按钮必须继续复用统一音频按钮组件
- **AND** 其视觉层级必须与当前页面已确认的纯 icon 音频按钮规则保持一致
